# Xibalba Pinball — Art Direction Guide

## Vision

Xibalba Pinball should feel like a premium collectible pinball machine, not a generic neon prototype.

The experience should evoke:
- an underworld temple
- ritual machinery
- obsidian stone
- aged gold ornament
- eclipse energy
- sacred geometry
- danger, mystery, and prestige

The table should feel expensive, memorable, and iconic.

---

## Primary Visual Direction

### Core theme
**Obsidian + Gold + Jade + Ember**

### Desired emotional tone
- ancient
- ceremonial
- dangerous
- mystical
- premium
- powerful
- collectible

### Avoid
- generic synthwave
- placeholder neon blue/pink
- overly noisy glow spam
- candy red / lime green
- “Christmas” red-green balance
- flat untextured bright colors

---

## Color Palette

### Base / materials
- **Obsidian Black** — `#0B0B0F`
- **Charcoal Stone** — `#1A1A1F`
- **Aged Gold** — `#C89B3C`
- **Deep Gold Shadow** — `#7A5A1E`
- **Bone / Pale Text** — `#E8DFC8`

### Feature / light colors
- **Jade Green** — `#1FA36B`
- **Bright Jade Glow** — `#5BE39D`
- **Ember Red** — `#B63A2B`
- **Blood Eclipse Red** — `#7E1F1F`

---

## Color Meaning System

- **Gold** = sacred structure, trim, architecture, premium machinery
- **Jade Green** = progress, activated features, mystical energy, ready states
- **Ember Red** = danger, jackpot, eclipse, high-energy states, multiball
- **Bone / Pale Text** = readable UI text and labels

---

## Table Material Rules

### Playfield
The playfield should read as dark obsidian stone with subtle depth and texture.

### Gold Elements
Gold should feel aged and premium, not bright flat yellow.
Use shadowing and contrast so it reads like metal trim or engraved sacred machinery.

### Glows
Glows should be selective and meaningful.
Use concentrated, tasteful halo effects rather than flooding the table with color.

---

## Feature Styling

### Temple Jackpot
The temple jackpot area is the hero centerpiece.
It should feel ceremonial, dangerous, and sacred.

Visual language:
- gold framing
- obsidian body
- jade idle/ready accents
- ember-red jackpot energy
- stronger emphasis than surrounding features

### Rollovers
Rollovers should feel like sacred glyph lanes.
Use:
- dark base
- gold framing
- jade lit progression
- clear readability

### Bumpers
Bumpers should feel like temple mechanisms:
- gold ring / sacred trim
- obsidian housing
- jade core glow
- red flash or stronger hit pulse on intense impacts if appropriate

### Slingshots
Slings should feel energetic but elegant:
- gold framing
- obsidian base
- jade glow
- ember flash on activation if useful

### Flippers
Flippers should no longer feel toy-like blue/pink.
Preferred look:
- gold / obsidian body
- jade and ember inserts or edge accents
- readable but premium

### Target Banks
Target banks should feel like carved or embedded temple markers:
- dark base
- gold edges
- lit accents tied to progress/readiness

### Eclipse Multiball
Use stronger ember-red + gold energy.
This state should feel powerful and dangerous.

### Eclipse Ready
Use jade + gold.
This state should feel charged and special without overpowering multiball visuals.

---

## UI Styling

UI should feel polished and integrated with the table theme.

### Score / High Score / State
- readable on mobile
- pale / bone text
- subtle gold accenting
- mode colors reflect game state

### Start Screen
Should feel premium and intentional:
- title with presence
- clean subtitle
- polished prompt styling
- consistent palette with the table

### Popups
Large popups should feel impactful and premium:
- TEMPLE JACKPOT
- ECLIPSE READY
- ECLIPSE MULTIBALL
- BALL SAVE

Use palette-appropriate accents:
- jackpot = ember + gold
- ready = jade + gold
- multiball = ember + eclipse energy

---

## Readability Rules

- The ball must remain highly visible against the darker playfield.
- Important shot lanes must stay readable.
- Gameplay clarity is more important than decoration.
- UI should not overwhelm the playfield.
- Mobile readability must be preserved.

---

## Implementation Constraints

### Do not change
- table geometry
- physics
- flipper tuning
- shot routes
- shooter lane
- drain/outlane behavior
- rollover rules
- jackpot rules
- multiball rules
- touch controls

### Focus on
- color system
- materials
- visual identity
- clarity
- premium presentation
- consistent style language

---

## Final Quality Bar

The result should feel:
- less like a prototype
- less like “AI neon pinball”
- more like a premium original machine someone would remember

If a choice is unclear, prefer:
**obsidian + aged gold base, jade for progress, ember for danger/jackpot/eclipse.**
