# Tuning Log

Use this file to record each tuning pass.

## Format

### Date / Pass Name

Changed:
- value changed
- geometry changed
- system changed

Result:
- what improved
- what got worse
- whether to keep or revert

## Known Good Checkpoint

Working:
- shooter lane exits
- flippers work
- reset/debug work
- ball can bounce around
- jackpot/rollovers exist
- anti-stuck helpers exist

Problem:
- upper playfield is too hard to reach
